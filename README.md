# Congressional Travel Explorer

## To setup app:

```bash
# need python 3.11 because only older version works with gensim
pyenv install 3.11
pyenv local 3.11
# create venv
python3 -m venv venv
# activate venv
source venv/bin/activate
# install requirements
pip install -r requirements.txt
```

## To run:

```bash
python app.py
```
